# First PR Practice

A tiny sandbox repo for practicing the GitHub pull request workflow:
fork/branch, commit, push, and open a PR.

## What's here

- `src/sum.js` — a small `sum` function
- `test/sum.test.js` — a test for it

## Usage

```bash
node -e "console.log(require('./src/sum').sum(2, 3))"
```
